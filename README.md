# Billboard Hot 100 Number Ones Research
Dorian Ramey

## Description
In this project, I create and explore three research questions based on the Billboard Hot 100 Number Ones dataset found here: https://github.com/rfordatascience/tidytuesday/tree/main/data/2025/2025
08-26

The goal was to better understand music classification and what makes a song successful, as well as determine if there have been changes over time. 
To achieve this, I created multiple visualization and models to explore relationships between features. With these, it is clearer to see the results 
to the research questions and explain these results to others.

This project was completed in R to allow for the model creation and graphical representation. The report was written in Overleaf to allow for LATEX representations.

## Contents
The file Final_Project_391.Rmd contains the code to create the visualizations and models that are used to answer the research questions.
STAT_391_Final_Project_Report.pdf contains a report outlining the project goals, questions, methods, and results.
